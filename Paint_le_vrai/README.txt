Pour utiliser ce programme, il va déja falloir compilier tous les codes sources (javac NomDuFichier.java).
Quand tout sera compiler il faudra executer Test.java (le main ce trouve dedans), donc faire un java Test dans le terminal.

Quand la fenêtre s'ouvre il faut impérativement choisir une couleur parce qu'elle est en blanc par défaut (onglet Couleurs).
Ensuite le dessin des formes ce fait en 2 cliques ( il faut ne faut pas cliquer trop rapidement sinon le programme n'enregistre pas le 2ème clique).
Ne pas oublier de valider si l'on veut pouvoir afficher le texte (dans l'onglet texte).
Le crayon s'utilise avec un appui long sur la souris.
La gomme n'est pas centrée sur la pointe de la souris mias plutôt vers le bas.
Le bouton Supprimer retir tout les dessins du panel alors que le bouton retour retire le dernier dessin du panel.
Si le remplissage est actif il sera mis en couleur (celle choisis).
Si on veut dessiner X fois la même forme il faudra d'abord appuyer sur le bouton, ensuite dessiner et refaire ça X fois (Appuyer sur un bouton pour dessiner une forme ne permettra d'en dessiner qu'une seule).
